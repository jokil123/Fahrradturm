\subsection{Github}
GitHub ist eine Online-Plattform für die Versionsverwaltung von Softwareprojekten. Es bietet eine Möglichkeit für Entwickler, den Code online zu teilen und zu speichern, um effektiv mit anderen zusammenzuarbeiten und Änderungen an Projekten zu verwalten.\cite{githuberklaerung} \\
GitHub wurde bei der Diplomarbeit verwendet, um den Code der App und des Controllers und die Dokumentation der Arbeit zu teilen und eine gemeinsame Zusammenarbeit zu ermöglichen.\\
